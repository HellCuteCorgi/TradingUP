Trading application
===============================================================================

Trading application system for your trading house. It provides
complete storage of product data and a fairly intuitive interface for ordinary users.

Installation Notes:
------------------

1. Download files from github.
2. Open .exe file and installation application.
3. Wait for the download to finish.
4. Open the installed application and start using.


Приложение для торгового дома
===============================================================================

Система торгового приложения для вашего торгового дома. Оно обеспечивает
полное хранение данных о товарах и имеет довольно интуитивно-понятный, для обычных пользователей, интерфейс.

Указания по установке:
------------------

1. Скачайте файлы с github.
2. Откройте .exe-файл и установите приложение.
3. Дождитесь завершения загрузки.
4. Откройте установленное приложение и начинайте использовать.
